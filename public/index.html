Bienvenido a mi Rest Server

Descripción

Servidor creado con Express, JWT, MongoDB

Con este Rest-Server puedes crear un usuario mediante la API o mediante Google Sing-in

Al autenticarte con tu usuario tambien puedes acceder a las funcionalidades de crear, editar, obtener y eliminar productos y sus respectivas categorias.

Ademas de poder subir y obtener imagenes de los productos o del perfil del usuario.
